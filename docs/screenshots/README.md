# Screenshots Directory

This directory contains screenshots for the BiR User Guide.

## Screenshot Files (8 total)

The following 8 screenshots should be placed in this directory:

1. **music-search-rock.png** - Music search results for "rock" showing band cards with similar bands feature
2. **music-similar-bands.png** - Similar bands feature showing recommendations (extracted from music search)
3. **natural-language-search.png** - Natural language search interface showing "music trends in Germany"
4. **spark-compare-interface.png** - Spark comparison tool interface for comparing countries
5. **comparative-analysis-results.png** - Comparative analysis results dashboard
6. **spark-compare-detailed.png** - Detailed side-by-side comparison view (United States vs United Kingdom)
7. **artwork-search-impressionism.png** - Fine Arts search results for "Impressionism" showing artwork cards
8. **network-graph-postimpressionism.png** - Network graph showing Post-impressionism movement (or Impressionism)
9. **network-graph-impressionism.png** - Network graph with Impressionism selected (optional, if different from above)

## Usage

These screenshots are referenced in `user-guide.html`. Make sure all image files are placed in this directory with the exact filenames listed above.

**Note:** Some screenshots may be reused in multiple sections (e.g., in case studies).
