1. Introduction

Welcome to BiR (Big Data Retriever), a web application that allows you to explore music bands and fine arts artworks from Wikidata. This guide will help you understand how to use the application effectively.

BiR provides two main domains:

  • Music Domain: Discover music bands, their genres, influences, and relationships
  • Fine Arts Domain: Explore artworks, artists, movements, and their connections

2. Getting Started

2.1 Accessing the Application

To access BiR, open your web browser and navigate to:

http://localhost:5173

You should see the BiR Analytics homepage with two tabs: Music and Fine Arts.

[Screenshot 1: BiR Homepage showing Music and Fine Arts tabs]

2.2 Interface Overview

The BiR interface consists of:

  • Header: Application title and tab navigation
  • Search Section: Search for bands or artworks
  • Statistics Panel: Global statistics and analytics
  • Results Grid: Display of search results
  • Visualization Tools: Network graphs and charts

Step 1: Click on the Music tab in the header

Step 2: In the "Entity Search" section, enter a band name or genre (e.g., "rock", "Beatles")

Step 3: Click the Search button or press Enter

The results will display matching bands in cards showing:

The Global Stats panel on the left shows real-time statistics from the Knowledge Graph:

4.6 Sharing Artworks via URL

Each artwork in the Fine Arts domain has a unique, shareable URL that can be used to access the resource directly. The URL format is:

http://localhost:5173/resource/art/{artwork-id}

Tip: You can copy the artwork's Wikidata URI or use the generated resource URL to share specific artworks with others. The URL contains all the information needed to identify and retrieve the artwork from the Knowledge Graph.

Case Study 1: Discovering UK Rock Bands from the 1980s

Objective: Find rock bands from the United Kingdom that were active in the 1980s and discover their influences.

Steps:

  1. Navigate to the Music tab
  2. In the Natural Language Search section, enter: "rock bands from UK in the 1980s"
  3. Click Search
  4. Review the results showing bands like The Smiths, The Cure, etc.
  5. Click on a band card and use "Find Similar" to discover related bands

The search returns multiple UK rock bands from the 1980s, including their genres, founding years, and countries. The similarity feature helps discover bands with similar musical styles or influences.

7. Tips and Tricks

7.1 Search Tips

  • Use partial names - the search is case-insensitive and matches partial strings
  • Try genre names for broader results (e.g., "rock", "jazz")
  • For artworks, search by artist name, movement, or artwork type

7.2 Navigation Tips

  • Use tabs to switch between Music and Fine Arts domains
  • Scroll down to see all available features
  • Click on cards to expand and see more details

7.3 Visualization Tips

  • Network graphs are interactive - zoom and pan to explore
  • Click on nodes to see connections
  • Use the comparison tool to analyze trends

8. Troubleshooting

8.1 Common Issues

No Search Results

Problem: Search returns no results

Solution:

  • Check if the ETL pipeline has completed (wait 2-5 minutes after startup)
  • Try a different search term
  • Check browser console for errors

Statistics Not Loading

Problem: Statistics panel shows no data

Solution:

  • Refresh the page
  • Check if Fuseki Knowledge Graph is running
  • Verify API Gateway is accessible

Network Graph Not Displaying

Problem: Network graph doesn't appear

Solution:

  • Ensure you have search results first
  • Check browser console for JavaScript errors
  • Try refreshing the page
